# 🐍 Mundo Python - Curso em Vídeo

Este repositório é dedicado ao meu aprendizado de Python 3, seguindo a trilha do professor **Gustavo Guanabara** no [Curso em Vídeo](https://www.cursoemvideo.com/). Aqui documento todos os meus exercícios e desafios resolvidos diariamente.
